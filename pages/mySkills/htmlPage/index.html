HTML

HTML (Hyper Text Markup Language) is a language that the web browser uses to show what you are seeing right now. For example, when you right click anywhere on this page and then click inspect.

Everything under < html > is the HTML code. It represents the structure of a web page and consist of elements that tell the web browser what to display on the webpage.

HTML Elements/Tags

An HTML element is evverything from the start and end tag.

Example:

In every HTML file will have a < html > element that end with < /html > which is the whole html document. In the element will be another html element, < body > and end with the closing tag < /body > that represent the body of the document. There are many different type of elements and tags so click here for more info.

Attributes

HTML attributes give information about HTML elements. All HTML elements can have attributes and are always stated in the start tag. Attributes like class's and id's are used to point to a specific element in css style sheet. Class atrributes are the same as id's but can be used for multiple elements, when id attributes are only used for one element where no other element can have a the same id.

Inline and Block-level Elements

There are also inline and block elements. Inline elements are elements that take up as much space as needed and does not go into the next line. Block-level elements always start on a new line and takes up the full width that is available.

keep in mind that some tags might not have an ending tag. Those are called self closing tags, and some tags might not have any content.

Examples of Block-level elements:

Start tag Content End tag
< h1 > Header tag < /h1 >
< div > Divider tag < /div >
< p > Paragraph tag < /p >

Examples of inline elements:

Start tag Content End tag
< a href="" > link tag < /a >
< img > This is a self closig tag with no content*
< i > Italic tag < /i >
