CSS

CSS (Cascayding Style Sheet) is code we use to style a Web page. It describes how HTML elements are to be displayed on screen including the design, layout and display for different devices and screen sizes. You would make a seperate file for css and work on your css there rather than in your html file otherwise the html file would be very big and would get confusing to look at. How you would link the html and css together is to link your style sheet in the head of your html document.

For the elements that you want to style and position you would add either a class or id attribute to the start tag and name it. So when you go into your css file you can pin point that element.

Positioning

  • static; (normal flow) html elemtns are positioned static by default.
  • relative; (relative to it's normal position) Top, right, bottom, and left properties of a relatively-positioned element will be adjusted away from its normal position.
  • absolute; (relative to it's nearest positioned parent)
  • fixed; (fixed when scrolling) The fixed position is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.
  • sticky; (switches between relative to fixed based on scroll postion)
