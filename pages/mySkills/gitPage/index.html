Git / GitHub

GitHub is a code hosting platform for collaboration. It lets you and others work together on projects from anywhere. Team members can work on files and easily merge their changes in with the master branch of the project by pushing and pulling files form github.

Git is a type of system that makes it easier to track changes to files. For example, when you edit a file, git can help you determine exactly what changed, who changed it, and why.

You would use terminal and list of commmands to add those files to commit. You could think of adding files to commit as adding things into a basket. Then you would push those files onto github like sending the basket onto a shelf where you can also pull files down.
