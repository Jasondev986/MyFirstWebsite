Git / GitHub

GitHub is a code hosting platform for collaboration. It lets you and others work together on projects from anywhere. Team members can work on files and easily merge their changes in with the master branch of the project by pushing and pulling files form github.

Git is a type of system that makes it easier to track changes to files. For example, when you edit a file, git can help you determine exactly what changed, who changed it, and why.

You would use terminal and list of commmands to add those files to commit. You could think of adding files to commit as adding things into a basket. Then you would push those files onto github like sending the basket onto a shelf where you can also pull files down.

Steps:

Let's say you have been working on your first website and want to push all the files to github.

Terminal:

List of Commands Used:

  • ls "to list folders and files"
  • cd MyFirstWebsite "file you want to commit"
  • git add . "to add all the files to the commit"
  • git commit -m "type message in quoutations"
  • git log "to see the all the commited files and folders"
  • git push "to push all the files and folders to gitgub"
  • clear "to clear everything"

First I typed "ls" to list all the folders in my home tab. Next I typed "cd workspace" change directory to go into the folder workspace. Then "cd MyfirstWebsite" change directory into that folder. Then "ls" to list everything in that folder. I then typed "git add ." to add all the files to commit. Next I typed "git status" to see everything that i'm going to commit as you can see the "changes to be committed:". You will also have to put a message for the commit in quoutations for all the files and modified files.

Finally I would type in the command "git push" to push the files to github. In a few seconds the files would be pushed into your repository where other people can pull your changes.
